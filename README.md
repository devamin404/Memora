## 🛠️ Tech Stack & Architecture

### Frontend (Current Phase)

- **Framework:** React.js (Built using **Vite** for lightning-fast performance)
- **Styling:** Pure **CSS Modules** (Leveraging modern **Flexbox** and **CSS Grid** for layouts)
- **Navigation:** **React Router DOM** (For seamless Single Page Application routing)
- **Icons:** **Phosphor Icons**

### Backend & Database (Planned Roadmap)

- **Server:** Node.js & Express.js
- **Database:** MongoDB (Mongoose ODM)
- **Cloud Storage:** Cloudinary or AWS S3 (For robust image, video, and audio file uploads)

---

## ✨ Key Features Implemented So Far

- **Premium Dashboard UI:** A clean, minimal, and highly professional workspace layout.
- **Dynamic Stats Overview:** Live tracking counters for total Folders, Memories, Photos, Videos, and Audios (ready for backend API integration).
- **Custom Grid Layout:** Beautiful folder cards featuring smooth CSS scale-up hover animations and glass-morphism badges.
- **Recent Memories Section:** A sleek, unified list view displaying the latest journal entries with clean CSS text truncation (`...` ellipsis).
- **Interactive UI Elements:** Advanced custom cursor behaviors (`user-select: none`, `cursor: pointer`) and layout alignments utilizing modern CSS shorthand methods (`display: grid`, `place-items: center`).
